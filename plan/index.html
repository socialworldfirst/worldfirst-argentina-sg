How to run it live

Pre-approve the per-day rate and we simply keep the meter running each day they're alive, no re-approval per round. The two fixed pillars (9 upfront days + the 5-day last-dance tail) are committed; everything past Jul 3 is day-by-day and stops the moment they exit. Optional refinement: if a flat rate feels too even, we can let the deep-round days (QF/SF/Final) run at a higher per-day rate, the model already isolates them.
